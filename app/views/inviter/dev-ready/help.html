Help using Apply for Care ID

Help an applicant

  1. View the applicant support page for advice with common issues
  2. If they are experiencing a technical issue which is preventing them from completing their application, go to the NHS Digital Customer Service Portal. Include the application ID and full details of the issue. Alternatively, email ssd.nationalservicedesk@nhs.net.
