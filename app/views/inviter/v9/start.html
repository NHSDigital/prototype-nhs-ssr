Apply for Care ID is

Use this service to:

  • enter applicant details to invite them to apply for an NHS Care ID
  • resend, edit and cancel applications your organisation has raised
  • view applications where duplicate profiles have been found
  • view approved and rejected applications

You will need Windows Hello, your Security Key or Smartcard with the right permissions and roles to manage NHS Care ID applications.
