Find out if you already have an NHS Care ID profile

To check if you already have an NHS Care Identity profile, we'll:

  • send a security code to your mobile phone to check that the number we have for you is correct and to make sure that you can access it
  • ask for your full name, date of birth and national insurance number.

If you do not have a profile, you will need to verify your identity to get one.
