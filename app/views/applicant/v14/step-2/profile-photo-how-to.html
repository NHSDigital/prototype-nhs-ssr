How to get a passport style photo of your face

Use a photo you already have

If you already have a suitable photo, you'll need to:

  • make sure you can access the photo on this device
  • upload it using the web form

Take a photo

If you do not have a photo already, you will need to:

  • take a new photo using a smartphone or tablet. You may need someone to help you take it
  • make sure you can access it on this device
  • upload it using the web form

If you cannot take a photo now, you can return to your application later by using the link in your invite email.
