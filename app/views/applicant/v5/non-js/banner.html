How to enable Javascript

To enable Javascript on your default browser follow the steps below:

  1. From your browser’s menu (shown as three dots) select “Settings”.
  2. Search for Javascript using the search box.
  3. Find the ‘Javascript’ section.
  4. Set Javascript to ‘Allowed’ as default

Once Javascript is enabled go back to your email and retry the link.

If you are unable to change your Javascript settings, please contact your local IT support for help.
