Apply for Care ID

Apply for Care ID is a secure online ID verification service for NHS users to submit and verify their documents online.

Once their identity has been verified, a profile will be created for them in Care Identity Service (CIS).

Use the service to:

  • enter applicant details to invite them to apply
  • update Care ID applications you have submitted
  • check the progress of Care ID applications
Start

Before you start

You will need Windows Hello, a Security Key or smartcard with one of these roles assigned to use Apply for Care ID:

  • R5080: Registration Authority manager
  • R5090: Registration Authority agent
  • B1300: Sponsor

For support using Apply for Care ID view the Apply for Care ID guidance.
